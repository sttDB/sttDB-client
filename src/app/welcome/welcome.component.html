About the project

sttDB is a transcriptome database from telencephalon and muscle tissues of brown trout (Salmo trutta trutta).

Show more

This database includes the assemblies and annotation of our original de novo transcriptome data and genomic resources generated from muscle and telencephalon tissues using NGS technologies (RNA-seq). The information provided by this resource includes Gene Ontology terms, KEGG pathway assignation, Protein Family annotation and SNP discovery. It also provides a BLAST server for matching user-provided sequences to our databases. Data provided by sttDB can be downloaded using the “Downloads” option in the menu bar. Please cite sttDB web site for publication purposes.

Protein families

InterPro is a resource that provides functional analysis of protein sequences by classifying them into families. Search the sequences of our database that matches against the InterPro protein signature databases.

Gene Ontology (GO)

The Gene Ontology (GO) project provides the most comprehensive resource currently available for computable knowledge regarding the functions of genes and gene products.

Show more

The GO terms describe gene function classified along three aspects: molecular function as molecular activities of gene products; cellular component relating where gene products are active and biological process as pathways and larger processes made up of the activities of multiple gene products.
